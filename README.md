# Email Spam Classification

Machine learning project that classifies emails as spam or ham using natural language processing techniques.

## Dataset
Email dataset containing labeled spam and non-spam messages.

## Approach
- Text preprocessing
- Feature extraction using TF-IDF
- Model training using machine learning classifiers

## Tech Stack
Python | Pandas | NumPy | Scikit-learn | NLP | Jupyter Notebook

## How to Run
pip install pandas numpy scikit-learn
jupyter notebook email-spam-classifier.ipynb
